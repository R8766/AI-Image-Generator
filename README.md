# AI-Image-Generator
🚀 Project Overview

The AI Image Generator is a web-based tool that transforms text prompts into AI-generated images. Users can describe their imagination in detail, and the application generates stunning visuals using advanced AI models.

This project demonstrates practical use of AI, JavaScript, and web development to create interactive, real-time image generation applications. It’s an excellent showcase for anyone interested in AI-powered frontend projects.

✨ Features

-Generate images from user text input
-Select AI model (Stable Diffusion XL or Stable Diffusion 3)
-Choose image count and aspect ratio
-Random prompt generator for inspiration
-Light/Dark theme toggle
-Download generated images directly

🛠️ Tech Stack

-Frontend: HTML, CSS, JavaScript
-AI/ML: Hugging Face Inference API
-Libraries: Font Awesome for icons
-Deployment: Browser-based, responsive UI

📥 Installation / Usage

1.Clone the repository:
git clone https://github.com/R8766/AI-Image-Generator.git

2.Navigate to the folder and open index.html in your browser.

3.Usage Steps:

-Click Get Started on the landing page
-Enter a text prompt describing your desired image
-Select AI model, image count, and aspect ratio
-Click Generate to see images appear in the gallery
-Click the download button to save any generated image
